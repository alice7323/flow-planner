# FlowPlanner

Your flexible daily planner web app - designed for people who love spontaneity and flexibility!

## 🚀 View Live App

Once deployed, your app will be available at: https://alice7323.github.io/flow-planner/

## 📝 About

FlowPlanner is a simple, visual planner designed specifically for MBTI P-type personalities who:
- Prefer flexibility over rigid schedules
- Like to adjust plans on the go
- Need visual summaries to stay organized
- Want simple, stress-free planning

## ✨ Features

- ✅ Easy task input
- 📊 Visual progress tracking
- 🎯 Simple interface
- 💾 Auto-saves your plans
- 📱 Works on any device

## 🛠️ Tech Stack

- Pure HTML/CSS/JavaScript
- No dependencies
- LocalStorage for data persistence
- Responsive design
